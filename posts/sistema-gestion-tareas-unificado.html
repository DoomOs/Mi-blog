Sistema de Gestion de Tareas en Python con PyQt5, SQLite y FPDF

Posted on 17-03-2025 19:00 in Desarrollo de Software

Sistema de Gestión de Tareas en Python

En este tutorial se explica el desarrollo de un sistema de gestión de tareas diseñado para registrar, pausar, reanudar y finalizar tareas, además de generar reportes en PDF. La aplicación se ha implementado en un solo archivo para facilitar su empaquetado en un ejecutable (.exe) utilizando PyInstaller.

Nota: El código completo se ha subido a un repositorio en GitHub.

Características Principales

  • Registro y Gestión de Tareas:
    Permite iniciar tareas con estado en proceso o pausadas, registrando la hora de inicio y, en su caso, las pausas.

  • Control de Pausas y Reanudación:
    Se registran automáticamente los tiempos de pausa y se calcula el tiempo efectivo trabajado.

  • Generación de Reportes en PDF:
    Con FPDF, se pueden generar PDFs que resumen la información de cada tarea y los rangos de fechas seleccionados.

  • Interfaz Gráfica con PyQt5:
    Una GUI intuitiva que permite gestionar las tareas mediante pestañas: una para registrar, otra para administrar y una tercera para generar reportes.

  • Persistencia con SQLite:
    La base de datos se crea en un directorio específico del usuario, lo que asegura que la información se guarde entre sesiones.

Arquitectura del Sistema

El sistema sigue el patrón MVC (Modelo-Vista-Controlador) para separar la lógica de negocio, la presentación y el acceso a datos.

Modelo (Acceso a la Base de Datos)

Se utiliza SQLite para almacenar la información. Solo se crean dos tablas:

  • tasks: Guarda los datos de cada tarea (título, descripción, tiempos y estado).
  • pauses: Registra las pausas asociadas a cada tarea, con una relación de clave foránea.

Snippet de creación de tablas:

def create_tables(self):
    cursor = self.conn.cursor()
    # Tabla de tareas
    cursor.execute('''
        CREATE TABLE IF NOT EXISTS tasks (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            title TEXT NOT NULL,
            description TEXT,
            start_time TEXT,
            end_time TEXT,
            status TEXT
        )
    ''')
    # Tabla de pausas
    cursor.execute('''
        CREATE TABLE IF NOT EXISTS pauses (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            task_id INTEGER,
            pause_start TEXT,
            pause_end TEXT,
            FOREIGN KEY (task_id) REFERENCES tasks(id)
        )
    ''')
    self.conn.commit()

Controlador (Lógica de Negocio)

El controlador gestiona las operaciones fundamentales, tales como:

  • Agregar Tarea: Registra una nueva tarea y, si se inicia en pausa, también crea el registro de la pausa.
  • Pausar y Reanudar: Actualiza el estado de la tarea y gestiona los registros de pausas.
  • Finalizar Tarea: Marca una tarea como finalizada y cierra cualquier pausa abierta.
  • Generar PDFs: Produce documentos con el detalle de cada tarea o reportes de un rango de fechas.

Fragmento de código para agregar una tarea:

def add_task(self, title, description, state="en proceso"):
    now = datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    task_id = self.db.add_task(title, description, now, state)
    if state == "pausada":
        self.db.add_pause(task_id, now)
    self.view.show_message("Tarea iniciada con ID: " + str(task_id))
    self.view.refresh_task_table()

Vista (Interfaz Gráfica)

La interfaz se implementa con PyQt5 y está organizada en tres pestañas:

  • Registrar Tarea: Entrada de datos para crear nuevas tareas.
  • Gestión de Tareas: Visualización en tabla, con botones para pausar, reanudar y finalizar.
  • Reportes: Selección de fechas para generar reportes y ver resultados en pantalla.

Ejemplo de inicialización de la pestaña de registro:

def init_registrar_tab(self):
    self.registrar_tab = QWidget()
    layout = QVBoxLayout()
    self.title_input = QLineEdit()
    self.title_input.setPlaceholderText("Título de la tarea")
    layout.addWidget(QLabel("Título:"))
    layout.addWidget(self.title_input)
    self.registrar_tab.setLayout(layout)
    self.tabs.addTab(self.registrar_tab, "Registrar Tarea")

Empaquetado en un Ejecutable (.exe)

El código se encuentra en un único archivo, lo que simplifica su empaquetado. Se puede utilizar PyInstaller para generar un ejecutable de la siguiente manera: 

pyinstaller --noconsole --onefile --icon=icono.ico gestion_tareas.py
  • --noconsole: Oculta la consola.
  • --onefile: Genera un único ejecutable.
  • --icon: Permite agregar un ícono personalizado.

Conclusión

Este sistema de gestión de tareas combina Python, PyQt5, SQLite y FPDF para ofrecer una solución integral y sencilla de usar. La estructura en un solo archivo facilita la distribución y el empaquetado en un ejecutable, ideal para compartir y desplegar la aplicación sin complicaciones.
